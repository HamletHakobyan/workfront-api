<static> getInstance(config, returnNewInstance) → {Api}

Returns an Api instance. Creates a new one if no instance exists.
One can use this if Api is intended to be used as singleton.
Parameters:
Name Type Argument Description
config Object An object with the following keys:
url {String} - Required. An url to Workfront server (for example: http://localhost:8080)
version {String} - Optional. Which version of api to use. At the moment of writing can be 1.0, 2.0, 3.0, 4.0. Pass 'internal' to use Workfront internal API (this is the latest version, maybe unstable)
returnNewInstance Boolean <optional>
 If true, always creates a new instance
